# Контекст
Это атмосферный хоррор-прототип, который я разрабатываю в одиночку параллельно с учебой.

В проекте используется полностью Realtime освещение (URP). Я понимаю, что для мобильных платформ потребовалось бы запекание (Baking), но для ПК-прототипа я выбрал динамику ради атмосферы и скорости разработки.
Код написан просто и прямолинейно, чтобы быстрее реализовать механики.

# Про игру
Игрок идет по коридору отеля, слушая классическую музыку на плеере. Постепенно окружение меняется, и чувство безопасности разрушается.

Основная идея — создание напряженной атмосферы через окружение и звук, с минималистичным геймплеем.
